# visualizer-design

Codex-compatible SVG/HTML visual design skill for creating professional diagrams, charts, interactive widgets, maps, and lightweight illustrations.

This skill was adapted from a visualizer-style prompt into a Codex-native workflow. It guides Codex to deliver visuals through Mermaid, fenced SVG/HTML, local SVG files, local HTML files, or Data Analytics widgets when appropriate.

## What it covers

- Visual routing for diagrams, charts, maps, interactive explainers, and illustrations
- Flat design tokens, typography, spacing, and color ramps
- SVG structure, accessibility, arrows, node patterns, and layout rules
- Chart.js and D3 map guidance for local HTML artifacts
- Mermaid recommendations for ERDs and standard architecture diagrams

## Install

Copy this directory into:

```bash
~/.codex/skills/visualizer-design
```

Then restart Codex so it can discover the skill.

## Entry Point

Read `SKILL.md` first. Follow the quick reference table there to load only the relevant guide files for the requested visual.
